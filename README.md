### 起動方法

- フロントエンド

npm start

###　開発方針

1. MVPをWebで完成
- FastAPI＋Reactでバックエンド・フロントを形にする
- モバイル対応のUIを意識してレスポンシブに作る

2. モバイルアプリ化
- Expo（React Nativeの開発環境）をセットアップ
- 既存のReactコンポーネントをReact Native用に移植
- FastAPIと通信（API利用）

3. テスト
- Expo Goアプリで実機テスト
- iOS/Android両方で挙動確認

4. リリース準備
- Apple Developer Program（年間約12,000円）登録
- Google Play Developer（買い切り25ドル）登録
- アイコン・スクショ・説明文を準備

5. ストア申請
- iOSは審査が厳しい（1週間前後）
- Androidは比較的早い（1日程度）
